## **Install and Run Locally**

```
$ git clone https://github.com/Krushna-Kulkarni/geektrust_admin_ui
$ cd geektrust_admin_ui
$ npm install
$ npm start
```

## **Testing and Code Coverage**

```
for Testing
$ npm run test

for code coverage
$ jest --coverage

```
